이 글에서는 `viewBox`에 대해서 다루지 않는다.

### `width` 및 `height` 속성을 제거한다.

```html
<svg
  width="33"
  height="30"
  viewBox="0 0 33 30"
  fill="none"
  xmlns="http://www.w3.org/2000/svg"
>
  ...
</svg>
```

이를테면, 위와 같이 작성된 `svg`에서 아래와 같이 `width`와 `height`을 제거한다.

```html
<svg viewBox="0 0 33 30" fill="none" xmlns="http://www.w3.org/2000/svg">
  ...
</svg>
```

그리고 아래와 같이 `SVG`를 사용할 수 있다.

```html
<div className="xl:w-16 xl:h-16 md:w-10 md:h-10 w-4 h-4">
  <IconDelete className="w-full h-full" />
</div>
```
